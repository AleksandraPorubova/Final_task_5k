Команды для навигации и базовой работы с файлами в терминале macOS

КОМАНДА ФУНКЦИЯ
ls выводит список файлов и папок в текущей директории
ls -l покажет содержимое текущей директории в виде списка с дополнительной информацией
cd выполняет смену текущего каталога
cd.. переходит на уровень выше в иерархии каталогов
pwd отображает полный путь к текущей рабочей директории
file расскажет, что перед вами изображение, директория или текстовый документ, а также предоставит информацию о них
mv Перемещает файлы или директории. Сначала указывается переносимый файл, потом новый адрес. Эта же команда используется для переименования файла или директории. В этом случае сначала указывается старое имя, затем новое. При выполнении перемещения или переименования нужно убедиться, что вы находитесь в нужной директории, либо указать полный путь к папке.
rm удаляет файлы из файловой системы (без перемещения в корзину) без возможности восстановления
rm -r удаляет директории вместе с их содержимым из файловой системы (без перемещения в корзину) без возможности восстановления.
rmdir удаляет пустые директории
mkdir создает новую директорию
touch создает новый файл
cp копирует файлы или директории
cat Выводит содержимое текстового файла на экран. Также может соединять содержимое нескольких текстовых файлов в один новый. Данная команда может использоваться для создания нового текстового файла или добавления в него содержимого.
nano открывает текстовый редактор в терминале для редактирования файлов.
-h выдает справку о командах
exit выход из оболочки командной строки
